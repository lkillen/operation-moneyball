moneyball-pipeline
Purely mechanical · zero analytical responsibility · Bronze in, Silver parquet + Gold JSON out · event-driven · single control plane through dispatch.py
🔑 STORAGE_REPO_TOKEN — fine-grained PAT · read/write moneyball-storage
🔑 CF_R2_TOKEN — Cloudflare R2 API token
🔑 CF_R2_BUCKET_NAME — R2 bucket target
🔑 DISCORD_WEBHOOK_URL — passed to moneyball-notifications
🔑 CACHE_REFRESH_TOKEN — POSTs to Flask /cache/refresh after Gold deploy
🔑 FLASK_BASE_URL — Flask app URL for cache invalidation
GitHub Actions — single workflow
pipeline.yml — 6 jobs, one runner, shared state throughout
trigger: repository_dispatch from analytics · workflow_dispatch (manual — sport, league, season inputs)
Job 1 setup: checkout pipeline · clone analytics egress/ → temp/ (CSVs + sidecar manifests) · clone moneyball-storage · install deps
Job 2 run_pipeline: run dispatch.py · per-file loop · ingress → validate → health → transform → publish · outputs RunSummary
Job 3 commit_silver: git add/commit/push Silver parquet to moneyball-storage (if ≥1 file passed)
Job 4 deploy_gold: upload Gold JSON from gold_staging/ to Cloudflare R2 (if ≥1 file passed)
Job 4b invalidate_cache: POST to Flask /cache/refresh with token · clears lru_cache in place · no restart
Job 5 write_run_manifest: build + commit run manifest to moneyball-storage/runs/ · last confirmation everything landed
Job 6 notify: always fires · calls moneyball-notifications composite action · passes formatted RunSummary message
Core pipeline — dispatch is the single control plane
dispatch.py
single control plane · every stage is called by dispatch · every result flows back to dispatch · only dispatch talks to observability
flow: ingress.scan() → file list → for each file: ingress.run() → validate.run() → dispatch writes DQ report to health/ → transform.run() → publish.run() → build RunSummary → observability.collect() → return to workflow
failure on one file does not block others · partial success is a valid run outcome
temp/
Bronze · runner-ephemeral in production
copied fresh from egress/ each run

core/
DatasetPacket — dataframe + metadata (ingress → validate → transform)
TransformPacket — parquet bytes + JSON payload (transform → publish)
WriteMode enum · Stage enum · shared contracts · no logic
Pipeline stages — per file · called by dispatch
Ingress
scan() → file list from sidecar manifests
run(file) → verifies checksum · loads CSV
wraps as DatasetPacket · writes Bronze receipt
aborts if manifest missing or checksum mismatch
Validate
Bronze → Silver boundary · loads schema contract
populates write_mode, primary_keys, schema_version on packet
type coercion · DQ score · tags DatasetPacket
returns DQ report in StageResult → dispatch → health/
abort if empty · missing col · key null · threshold fail
Transform
DatasetPacket → TransformPacket
BATCH / APPEND / UPSERT on primary keys
parquet bytes (Silver) + Gold JSON from dataframe in memory
gold_validator checks Gold shape before packaging
Publish
TransformPacket → local filesystem
Silver → cloned moneyball-storage/parquet/
Gold → gold_staging/ on runner (picked up by Job 4)
atomic per file · reports success/failure to dispatch
Supporting modules
Schema

Health

pass/fail · DQ score · thresholds at time of run

Manifests
builds + writes run manifest to storage/runs/
one manifest per run · append only · never overwritten
written by Job 5 after deploy_gold · final confirmation
Observability
end-to-end monitor · passive · not in critical path · called only by dispatch via RunSummary
watches: stage durations · abort rates · row count trends · file size trends · write mode awareness (batch vs append)
silence detection: expected runs that didn't happen · datasets not updated in expected window
upstream trigger chain monitoring: detects when analytics pushed but pipeline never ran (egress/ commit newer than last run manifest) · loud failure alert via Discord
reads health/ for DQ drift detection · consumer of full run history
Tests

unit/ — per module · integration/ — stage interactions end to end · never touches real temp/ or moneyball-storage
